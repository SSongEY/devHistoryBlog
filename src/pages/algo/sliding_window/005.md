- 풀이

```java
public static int maxProfit(int[] prices) {
    int maxProfit = 0;
    int left = prices[0];
    
    for (int i = 1; i < prices.length; i++) {
        int current = prices[i];
        if (current < left) {
            left = current;
        } else {
            maxProfit = Math.max(current - left, maxProfit);
        }
    }
    
    return maxProfit;
}
```
