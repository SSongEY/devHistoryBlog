- 풀이

```java
public Roots findRoots(double a, double b, double c) {
    double sqrt = Math.sqrt(Math.pow(b, 2) - (4*a*c));

    double x1 = (-b + sqrt) / (2*a);
    double x2 = (-b - sqrt) / (2*a);
    
    return new Roots(x1, x2);
}
```
